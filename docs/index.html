Features

Everything You Need in One App

Eleven specialized tools designed with a premium dark interface, each optimized for a specific media workflow.

⚡

Encoding Presets

30+ one-click presets for Android, iPhone, YouTube, Instagram, TikTok, WhatsApp, professional editing, archival and more.

🔄

Format Conversion

Convert between any media format with full control over codecs, bitrate, resolution, frame rate, and quality settings.

✂️

Trim & Cut

Precision video trimming with stream copy support for instant, lossless cuts without re-encoding.

🎵

Audio Extraction

Extract and convert audio tracks to MP3, AAC, FLAC, WAV, Opus, and more with configurable quality.

🎨

Video Filters

Apply denoise, sharpen, blur, rotate, grayscale, sepia, vignette, stabilization and many other filters.

🔗

Merge & Concatenate

Join multiple video or audio files with stream copy or re-encoding. Drag-and-drop file ordering.

📐

Resize & Scale

Resize video to any resolution while preserving aspect ratio, perfect for social media targets.

📸

Screenshots

Extract individual frames or generate frame sequences at configurable intervals from any video.

📝

Metadata Editor

View and edit media file metadata tags — title, artist, album, year, and custom fields.

⌨️

Advanced Mode

Direct FFmpeg command-line input for power users who need full control over every parameter.

🔄

Auto Updates

Built-in update checker that connects to GitHub to notify you of new releases automatically.
